Project A: Excel 轉換至 NoSQL

  • 將 Microsoft Excel 資料庫，轉換至 NoSQL 的開發任務

Project Summary

  • Roxtal 是一家軟體服務公司，成立之初就全面導入 Microsoft Office 辦公室軟體

  • 客戶資料是以 Excel 維護，現在想將客戶資料「上雲端」，並製作 App 以隨時追蹤或查詢

  • 本專案在協助 Roxtal 公司，將客戶資料轉換為 NoSQL 資料庫。以便進行 RESTful Web Service 的開發，以及 App 的製作

Project A-1

  • 請使用 Node.js/mongoose 進行以下操作
  • 建立 vcard 資料庫
  • 建立 vcard.users collections
  • vcard.users 的 schema
    • name
    • phone
    • email
    • address
    • age

Project A-2

  • 請使用 Node.js/mongoose 進行以下操作
  • 建立 vcard.messages collection
  • vcard.messages 的 schema
    • userId（紀錄 ObjectId）
    • createdTime（現在的時間）
    • message（客戶回饋）

Project A-3

  • 請將 roxtal.xls 匯入 vcard.users collection
  • 方式不限、總計 1000 資料
  • 註：roxtal.xls 是程式產生的假資料，非真實資料

Project A-4

  • 希望加入 3 筆遺漏的資料
  • 使用 MongoDB script
  • 檔名：0010-add-admin-users.js
  • 加入 3 筆使用者資料：
    • Fred, 123-456-7890, fred@mandice.com
    • Hank, 321-000-6666, hank@moko365.com
    • Jollen, 666-123-7890, jollen@moko365.com

Project A-5

  • 公司想針對 45 歲以下的客戶進行行銷推廣
  • 使用 MongoDB script
  • 檔名：0011-find-most-value-person.js
  • 找出 18~30 歲的客戶，並放到 vcard.mvp collection
  • vcard.mvp 的 schema
    • userId（僅紀錄 ObjectId）

Project A-6

  • 將客戶的回饋儲存，留做後續追蹤
  • 使用 MongoDB script
  • 檔名：0012-post-message.js
  • 新增 3 筆訊息至 vcard.message
    • Dennis C. Molina: “this product is really good for me”
    • Xander A. Gordon: “can I can a red color one ?”
    • Rana R. Molina: “can I buy 10 and get 1 for free ?”
